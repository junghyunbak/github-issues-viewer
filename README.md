# github-issues-viewer

Github `issues` 탭에서 다음과 같은 불편한 점을 느껴, 편리하게 사용하고자 제작한 이슈 뷰어입니다.

- 이슈 간 빠른 전환의 어려움
- 사용성을 저해하는 다양한 정렬 & 필터링 요소 

레포지토리 이슈를 데이터베이스 삼아, 블로그처럼 작성하는 용도로 사용할 수도 있습니다.
